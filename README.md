# Laravel Inertia Package

This Laravel starter package consists of the following components:
- Laravel
- Inertia
- Fortify
- Vue
- tailwindcss

## Get started

Execute the following commands: 

    Rename the file .env.example to .env
    composer install 
    npm install 
    php artisan migrate 
    php artisan serve
